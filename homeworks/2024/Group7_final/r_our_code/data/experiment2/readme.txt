Experiment 2
List of files

bottomLDs_study2.mat: the Matlab output of the splitting procedure for bottom-ranked contours (data structure)
bottomTable_study2.mat:  the Matlab output of the splitting procedure for bottom-ranked contours (table)

Experiment2.m : Matlab file containing procedure for splitting and rendering of drawings by random forest model values 

Experiment2.R: summary aesthetic rating data analysis, ANOVA analysis for top vs bottom splits, mixed effects model with individual factors, and raw figures 

LD1.csv: datafile containing contour features and aesthetic ratings for split drawings.

LD2.csv: datafile containing individual characteristics of observers rating split drawings. 


scenesStatsLDs.mat: matlab data structure from mlvtoolbox.org containing contour properties of the Toronto Scenes Dataset

topLDs_study2.mat: the Matlab output of the splitting procedure for top-ranked contours (data structure)
topTable_study2.mat:  the Matlab output of the splitting procedure for top-ranked contours (table)

Predictions_Exp2.mat:  predicted aesthetic values for each drawing (used in variance partitioning analysis, reused in cross.csv file).
